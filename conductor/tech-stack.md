# 技術スタック: MatchNotes

## 1. コア技術

-   **プログラミング言語:** Dart
-   **アプリケーションフレームワーク:** Flutter (クロスプラットフォームモバイル開発用)
-   **状態管理:** Riverpod (リアクティブな状態管理用)
-   **データベースORM:** Drift (SQLiteデータベース操作用)
-   **データベースドライバ:** drift_sqflite (Drift用のSQLite実装)

## 2. 主要なライブラリと統合

-   **チャート作成:** `syncfusion_flutter_charts` (データ可視化用)
-   **CSV処理:** `csv` (CSVインポート/エクスポート機能用)
-   **ファイル選択:** `file_picker` (デバイスからのファイル選択用)
-   **Wakelock制御:** `wakelock_plus` (画面のスリープ防止用)
-   **ローカルストレージ:** `shared_preferences` (永続的なアプリケーション設定用)
-   **メディアストア操作:** `media_store_plus` (AndroidのMediaStoreとの対話用)
-   **ロギング:** `logging` (アプリケーションロギング用)
-   **クラッシュレポート:** `firebase_crashlytics` (クラッシュレポート用にFirebase Coreと統合)
-   **Firebase Core:** `firebase_core` (Firebaseサービスの基盤)
